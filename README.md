#### 프로젝트 설명
안드로이드 일정 관리 앱 

#### 구현한 기능
 - 캘린더 일정 추가 기능
 - 일정 목록 표시 기능

#### 사용한 기술 & 언어
 - Android - Java
 - SQLite
 - CalenderView, Timepicker
 - RecyclerView
